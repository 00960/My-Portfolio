ラーメンレビューアプリ

このラーメンレビューアプリは、ChatGPTなどのAIを活用しながら、SQLiteとPHPで構築したWebアプリです。
chobi.netのFREEプランを使い、データベースにMySQLを使わずにWebアプリを公開しています。
初学者としてPHPやSQLiteの基本構造を理解するため、AIと対話しながら必要なコードを生成・調整し、最終的には自分の手で環境構築・動作確認まで行いました。
今後は、AIをもっと効率よく活用しながら、新機能追加やリファクタリングに取り組んでいきたいと考えています。

🛠 使用技術： HTML / CSS / PHP / SQLite

📱 特徴：

  • ユーザー登録・ログイン機能
  • レビュー投稿・編集・削除機能
  • レスポンシブ対応でスマホでも見やすい設計
  • SQLiteベースで無料レンタルサーバー上に構築（chobi.net）
  • 画像アップロードやバリデーションも含めたセキュリティ対策

タイピング記録管理システム

職業訓練中に、学校側からこういったシステムが欲しいと言われたのがきっかけで制作したシステムです。
このシステムはAIを活用しながら、PHPとMySQLで作成したログイン機能付きのタイピング記録管理システムです。
ユーザーが1日1回だけタイピング文字数を記録し、過去の成績や最小・最大記録を管理できます。
また、職業訓練中での就職活動の記録する機能と管理者画面で全てのデータを確認できるページを追加で制作しました。

🛠 使用技術： PHP / MySQL / HTML / CSS

📱 特徴：

  • セキュアなユーザー認証（パスワードハッシュとセッション管理）
  • 1日1回の記録制限機能
  • 最小・最大記録の表示
  • スマホ対応レスポンシブデザイン
