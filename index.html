職業訓練で学んだ HTML / CSS / JavaScript を活かして、 モバイル対応のレスポンシブデザインとシンプルなUIを意識しています。

  • ホーム（トップページ）
  • メニュー
  • ギャラリー（店内や料理の写真）
  • アクセス（Googleマップ付き）

🛠 使用技術： HTML5 / CSS3 / JavaScript / GitHub Pages

📱 モバイル対応： スマートフォンやタブレットでも見やすく設計

🎨 デザインポイント：

  • 温かみのあるカフェをイメージした配色
  • シンプルでナビゲーションしやすい構成
  • ギャラリーは画像サイズを揃えて統一感を演出
